Production Backend Handbook - Part 2

This section covers request validation, standardized API responses, async error handling, and global error handling—the foundation before authentication.

Zod Validation

Why? Never trust client input. Validate every request before it reaches your controller.

Client ↓ Request Body ↓ Zod Schema ↓ Valid? ├── Yes → Controller └── No → 400 Response 
import { z } from "zod";

export const registerSchema = z.object({
  username: z.string().min(3).max(30),
  email: z.email(),
  password: z.string().min(8)
});

Validation Middleware

export const validate = (schema) => (req,res,next)=>{
  const result = schema.safeParse(req.body);

  if(!result.success){
    return res.status(400).json({
      success:false,
      errors:result.error.flatten()
    });
  }

  req.body = result.data;
  next();
};

Request Flow

Request ↓ express.json() ↓ validate(schema) ↓ Controller ↓ Service ↓ Database ↓ ApiResponse

ApiResponse Class

export class ApiResponse{
  constructor(statusCode,data,message="Success"){
    this.success = true;
    this.statusCode = statusCode;
    this.message = message;
    this.data = data;
  }
}

Example:

return res.status(201).json(
 new ApiResponse(
   201,
   user,
   "User registered successfully"
 )
);

ApiError Class

export class ApiError extends Error{
  constructor(statusCode,message){
    super(message);

    this.statusCode = statusCode;
    this.success = false;
  }
}

Async Handler

Instead of writing try/catch in every controller.

export const asyncHandler = (fn)=>
(req,res,next)=>{
  Promise.resolve(fn(req,res,next))
    .catch(next);
};

Controller Example

export const register = asyncHandler(async(req,res)=>{

 const user = await authService.register(req.body);

 return res.status(201).json(
   new ApiResponse(201,user,"Registered")
 );

});

Global Error Handler

export const globalErrorHandler =
(err,req,res,next)=>{

 return res.status(err.statusCode || 500).json({
   success:false,
   message:err.message || "Internal Server Error"
 });

};

Register Order

app.use(routes);

// LAST middleware
app.use(globalErrorHandler);

Error Flow

Controller ↓ throw ApiError ↓ asyncHandler ↓ globalErrorHandler ↓ JSON Response

Best Practices

  • Validate every input.
  • Never expose stack traces in production.
  • Use one response format everywhere.
  • Throw custom errors instead of strings.
  • Keep controllers thin; move business logic to services.
